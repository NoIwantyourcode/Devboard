# Devboard

This is a custom devboard that I am currently making, I features a RP2040 chip with 40 GPIO pins, I plan to integrate this devboard into a little Robot I am currently designing!

# Why I built this

I built this because I was extremely bored on one fine Sunday morning and decided to embark on this project to teach myself about robots and how to design them
